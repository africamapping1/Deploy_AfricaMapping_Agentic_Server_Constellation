# RealEstate AI Application

## Purpose

RealEstate AI is a business that uses the constellation to evaluate, manage, and operate real estate opportunities.

## Core Capability (MVP)

Input: property data  
Output:
- ROI estimate
- risk score
- rental potential

## Model

Activities → property inputs  
Projects → individual property investments  
Programs → portfolio strategy

## Vision

An autonomous real estate intelligence and operations platform.
