# RealEstate AI User Interface

## Purpose

Provides users with a portfolio view of their real estate assets.

## Core Views

- Portfolio summary
- Property list
- Property detail

## Data Sources

- Flow-03 (projects → properties)
- Flow-04 (programs → portfolio)
- Flow-05 (infrastructure context)

## Future Capabilities

- Mortgage tracking
- Payment initiation
- Portfolio analytics
- Risk visualization

## Principle

Users do not interact with infrastructure.
They interact with interpreted system state.
