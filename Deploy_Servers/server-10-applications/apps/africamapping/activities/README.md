# AfricaMapping Activities

This space holds business activity requests originating from AfricaMapping.

Flow-02 uses this space as the first application-driven intake path into the constellation.

## Purpose

To allow AfricaMapping business activity to enter the platform through a governed application path.

## Flow

activity intake
→ bastion receives
→ app processes
→ db stores
→ monitoring observes
→ Governance Loop evaluates
