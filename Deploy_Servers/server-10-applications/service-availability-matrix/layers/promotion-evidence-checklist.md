# Promotion Evidence Checklist

## Purpose

This layer defines the minimum evidence that should be checked before a matrix pattern is treated as a serious promotion candidate.

It exists so upward promotion decisions stay disciplined.

This is the promotion-evidence layer of the Service Availability Matrix.

## Why This Exists

The matrix now recognizes many possible promotion candidates.

But not every repeated pattern deserves upward movement.

This layer creates a basic evidence checklist before stronger promotion thinking continues.

## Core Principle

A pattern should not be promoted because it is interesting.

It should be promoted because evidence is strong enough.

## Promotion Evidence Checklist

Before a pattern is treated as a serious promotion candidate, check:

### 1. Repetition Evidence
- does the pattern appear in more than one tenant
- or, if only one tenant, is it clearly a tenant-type pattern rather than a local accident

### 2. Comparability Evidence
- are the observed versions structurally comparable
- do they solve the same or similar problem

### 3. Maturity Evidence
- is the pattern more than placeholder-level
- is it documented or stronger

### 4. Utility Evidence
- would promotion improve onboarding, parity, or standardization clarity

### 5. Boundary Evidence
- does the pattern genuinely belong above tenant space
- or is it still mainly tenant-local

### 6. Stability Evidence
- does the pattern still seem useful after review
- or is it changing too quickly

## Suggested Evidence Outcomes

### Weak Evidence
Do not promote yet.

### Moderate Evidence
Keep under observation or mark as promotion candidate.

### Strong Evidence
Consider stronger promotion review.

### Cross-Tenant Strong Evidence
Consider standardization or registration-default relevance.

## What This Checklist Helps Prevent

It helps prevent:
- weak patterns being promoted too early
- tenant-local habits being mistaken for platform truth
- premature standardization
- noisy bundle growth

## Final Principle

Promotion should follow evidence, not excitement.

This layer exists so upward movement stays earned and explainable.
