# Server 10 — Applications

## Purpose

Server 10 is the application layer of the constellation.

It hosts business-facing applications that consume infrastructure without modifying or destabilizing it.

## Principles

- applications must remain modular
- applications must remain separated from infrastructure internals
- applications interact through defined interfaces
- applications are governed by the Governance Loop

## Role in the Stack

Deploy Group → Infrastructure → Applications → Business → Customers

## First Consumer

The first business consumer in this layer is AfricaMapping.
