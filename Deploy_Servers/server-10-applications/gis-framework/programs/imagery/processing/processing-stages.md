# Imagery Processing Stages

## Stage 1
Use external imagery sources for immediate display and testing.

## Stage 2
Ingest raw imagery into internal workflows.

## Stage 3
Process imagery:
- reproject
- clip
- enhance
- prepare for downstream use

## Stage 4
Generate tiles or other display-ready imagery products.

## Stage 5
Serve imagery to:
- display engine
- foundational GIS programs
- composite GIS programs
