# Server 07 - AI Orchestrator

## Purpose

This server functions as the deliberative, governance, and routing layer of the constellation.

It is responsible for examining proposals, weighing impact, enforcing policy, routing decisions, and coordinating approved action.

## Role in Lifecycle

This server is central to the following stages:

- examine
- judge
- approve/reject
- governance enforcement
- decision routing

## Responsibility

This is not only a technical orchestrator.
It is also the constellation's weighing and judgment layer.

It compares new proposals against:
- existing structure
- policy
- compatibility
- historical decisions
- operational impact

## Principle

Nothing significant should proceed to execution without passing through governed review.
