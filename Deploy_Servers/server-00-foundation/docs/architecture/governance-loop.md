# Governance Loop

## Definition

The Governance Loop is the core cycle that ensures controlled, explainable, and validated evolution of the constellation.

## Components

- Narrator → observes, explains, records truth
- Strategist → interprets state and suggests direction
- Governor → decides and enforces policy

## Flow

System State → Narrator → Strategist → Governor → Decision

## Purpose

- ensure every change is reasoned
- ensure system behavior is understood
- ensure decisions are governed
- maintain alignment between intent, execution, and outcome

## Principle

No meaningful change occurs without passing through the Governance Loop.
