# Constellation Product Voice

The Agentic Server Constellation is a governed, self-evolving infrastructure system.

It ensures that every change:
- is evaluated
- is routed through defined roles
- is observed
- contributes to future improvement

This system is not static infrastructure.

It is a living constellation of servers that:
- receive
- examine
- decide
- act
- learn

## Core Principle

Deployment is not the end.

It is the beginning of system life.

## Narrator Role

The narrator explains:
- what the system is
- why it exists
- how it evolves
- how it should be used

It serves as:
- system memory
- product explainer
- operator guide
- sales voice

The narrator does not invent truth.

It derives truth from:
- system structure
- flows
- state
- policies
