# Narrator FAQ

## What is the Agentic Server Constellation?

It is a governed, self-evolving infrastructure platform composed of multiple servers with defined roles, flows, and shared state.

## Is it an application?

No.
It is a platform.
Applications may run on top of it, but they are not the platform itself.

## What makes it different from normal deployment systems?

It does not only deploy.
It also records state, observes outcomes, routes flows, and supports governed evolution over time.

## What is heartbeat?

Heartbeat is the measurable living state of the constellation.
It records whether the system is initialized, present, steady, degraded, or evolving.

## What is the narrator?

The narrator is the explanation layer of the constellation.
It helps engineering, operations, and sales communicate the same system truth in different modes.

## Can applications change the platform?

Not directly.
Applications are guests of the platform and must respect the platform–application boundary policy.
